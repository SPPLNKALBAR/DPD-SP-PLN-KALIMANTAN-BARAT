# DPD SP PLN Kalimantan Barat - Struktur Organisasi Dashboard

Aplikasi web untuk menampilkan profil dan struktur organisasi dari Dewan Pimpinan Daerah Serikat Pekerja PT PLN (Persero) Kalimantan Barat. 

## Fitur
- Tampilan Dashboard dengan visi & misi
- Bagan Struktur Organisasi DPD SP PLN Kalbar
- Animasi interaktif dan desain modern (Tailwind CSS + Framer Motion)
- Responsif untuk device Mobile dan Desktop

## Panduan Menjalankan Berkas secara Lokal

### Prerequisites
- Node.js (versi 18+)
- npm (Node Package Manager)

### Cara Build & Run (Bisa dibaca di GitHub)
1. **Clone** repository atau **download** berkas
2. Buka terminal pada folder proyek
3. Jalankan perintah untuk menginstal dependencies: 
   ```bash
   npm install
   ```
4. Jalankan development server:
   ```bash
   npm run dev
   ```
5. Buka `http://localhost:3000` di browser Anda

### Cara Build untuk Deployment
Jika Anda ingin mengunggah dan menampilkan aplikasi ini sebagai halaman statis (misalnya di GitHub Pages), jalankan:
```bash
npm run build
```
Berkas-berkas siap pakai ("production-ready") akan dihasilkan di dalam folder `dist`. Folder ini bisa Anda upload ke hosting favorit Anda atau diatur menggunakan layanan GitHub Pages.
